# Style Guide

Recipe Box is a warm, cozy, and inviting personal recipe manager. The design should feel like a well-loved kitchen — approachable, organized, and comfortable. Nothing should feel cold, corporate, or overwhelming.

In regards to color, create an application that follows the color schema already created. There is a .png file under @./skeleton called `design_colors.png`. I do not want you to deviate or make any changes to the color schema. As for the design, keep the UI clean and uncluttered — white space is your friend.

All destructive actions (delete, remove) must require user confirmation before proceeding.

Error messages should be clear, human, and never show raw technical details to the user

Consistency is key — buttons, cards, and forms should look the same across every page
